# ZimRental

**ZimRental** is a modern rental housing platform designed for Zimbabwe, connecting tenants and landlords to simplify rental property discovery, bookings, and payments. Our mission is to make housing search **transparent, trustworthy, and accessible**.

## Project Overview

In Zimbabwe, finding rental accommodation is often **slow, unreliable, and risky**, relying on WhatsApp groups, word-of-mouth, or agents. **ZimRental** solves this by:

- Providing verified property listings in one platform  
- Allowing tenants and landlords to communicate directly  
- Offering secure payments and deposit handling  
- Displaying verified reviews to build trust  

---

## Features

### Tenant Features
- Search properties by city, price, bedrooms, amenities, and verified listings  
- View property details with photos, videos, and floor plans  
- Book properties and pay deposits securely  
- Leave reviews for landlords  

### Landlord Features
- Create and manage property listings  
- Receive bookings and payments  
- Review tenants  
- Analytics dashboard for listing performance  

### Admin Features
- Approve property and user verifications  
- Monitor reports and disputes  
- Manage platform content and user activities

## Tech Stack

- **Frontend:** Next.js  
- **Backend:** Node.js + Express  
- **Database:** MongoDB  
- **Authentication:** JWT + Email/Phone verification  
- **File Storage:** Cloudinary  
- **Payment Gateway:** Paynow / Ecocash / Bank Integration  
- **Deployment:** Vercel (frontend), Render (backend)
